// Libraries

#include <windows.h>  // core Windows API
#include <stdbool.h>  // bool type
#include <stdint.h>   // optional, for fixed-width integers

// Flow

1. Initialize the game
    - Create a memory pool sized: 192MB
    - Load the map, player, entities, stats
    - Check the CPU hz count
    - Start the timer based off CPU hz count
    - Make a window and draw everything there
2. Run the game
    - Wait
3. End the game
    - Return to 0 for Windows
